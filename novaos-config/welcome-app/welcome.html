Development Setup

1. Configure Git

Set your name and email so commits are correctly attributed:

git config --global user.name "Your Name"
git config --global user.email "you@school.edu"

2. Set up Python

Use a venv per project to avoid conflicts:

python3 -m venv ~/projects/myproject/venv
source ~/projects/myproject/venv/bin/activate

3. C/C++ Toolchain

GCC, GDB, Make, and CMake are pre-installed. Verify with gcc --version.
